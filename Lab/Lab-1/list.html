Types of Programming Language

  1. Procedure Oriented programming:
    1. C-Programming:-
      1. High level programming language
      2. General-purpose computer programming language
      3. Created in the 1970s by Dennis Ritchie
    2. QBASIC:
      1. QBASIC stands for Quick Beginners All-Purpose Symbolic Instruction Code
      2. High level programming language
      3. Developed by Microsoft Corporation in 1985
  2. Object Oriented Programming:
    1. C++:-
      1. High-level general-purpose programming language
      2. Develop by Danish computer scientist Bjarne Stroustrup
      3. Create as an extension of the C programming language, or "C with Classes"
    2. Java:-
      1. High-level programming language
      2. Developed by Sun Microsystems in the early 1990s
      3. Used to develop desktop and mobile applications, big data processing, etc.
